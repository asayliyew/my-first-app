Вариант 7: Конвертер RGB-цветов

Дано: RGB(64, 224, 208)

1. Переводим R = 64 в HEX:
   64 / 16 = 4, остаток 0 → цифры: 4 и 0 → HEX: 40

2. Переводим G = 224 в HEX:
   224 / 16 = 14 (это E в HEX), остаток 0 → цифры: E и 0 → HEX: E0

3. Переводим B = 208 в HEX:
   208 / 16 = 13 (это D в HEX), остаток 0 → цифры: D и 0 → HEX: D0

Итоговый HEX-цвет: #40E0D0

Проверка:
HEX #40E0D0 = 
R: 4*16 + 0 = 64 
G: 14*16 + 0 = 224 
B: 13*16 + 0 = 208
